# SOP 03 - 异常应急预案（Incident Runbook）

## 事件分级

- **P0**：资金安全风险（疑似盗用、异常提币、失控下单）
- **P1**：交易中断或对账不一致
- **P2**：性能退化或局部功能异常

## 响应规则

### P0

1. 立即暂停自动交易
2. 撤销未成交订单
3. 降杠杆/平高风险仓位
4. 切换 API Key 或紧急冻结权限
5. 记录完整时间线并保留日志证据

### P1

1. 降级到低频或只减仓模式
2. 切换备用交易所/备用节点
3. 手工对账并恢复一致性

### P2

1. 标记影响范围
2. 限流或降级相关策略
3. 排期修复并增加监控

## 复盘模板要点

- 触发原因（技术/策略/外部）
- 影响范围（资金、时间、系统）
- 处置动作与耗时
- 根因修复项和防复发措施
