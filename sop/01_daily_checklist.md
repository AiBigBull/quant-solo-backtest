# SOP 01 - 每日检查清单（开盘前 / 运行中 / 收盘后）

## A. 开始前（每天固定）

1. 检查交易所 API 连通性、限频状态
2. 检查账户余额、保证金率、未平仓风险
3. 检查策略开关、参数版本、风控阈值版本
4. 检查数据流是否正常（延迟、缺失、异常）

## B. 运行中（每 15-60 分钟）

1. 监控滑点、拒单率、成交率
2. 监控回撤、杠杆、仓位集中度
3. 监控异常事件：交易所状态变化、订单积压
4. 触发阈值时执行风控动作并记录

## C. 收盘后（每日复盘）

1. 输出净值和收益归因（策略收益 vs 成本）
2. 核对订单与成交、手续费、资金费率
3. 记录当天异常与处理动作
4. 形成次日待办（优化项/故障修复/策略调整）
