# SOP 02 - 策略生命周期（研究 -> 仿真 -> 小资金 -> 扩容）

## 阶段 1：研究

- 形成策略假设与可检验指标
- 完成含成本回测（手续费、滑点、资金费率）
- 完成样本外与参数稳健性测试

通过条件：
- 成本后收益为正
- 最大回撤在可接受范围
- 参数微调下表现不崩溃

## 阶段 2：仿真

- 接入实时行情与仿真撮合
- 验证信号时效性与执行链路
- 检查风控拦截与告警是否生效

通过条件：
- 连续运行稳定（建议 >= 7 天）
- 风控触发逻辑符合预期

## 阶段 3：小资金实盘

- 用总资金的 1%-5% 上线
- 严格限制最大持仓和最大亏损
- 每日复盘实际滑点与回测偏差

通过条件：
- 实盘表现与仿真偏差在阈值内
- 无重大执行异常

## 阶段 4：逐步扩容

- 分档放大：5% -> 10% -> 20% ...
- 每档至少观察一个完整市场周期
- 任一档触发异常则退回上一档
